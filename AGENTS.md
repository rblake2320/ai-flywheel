# Agent Instructions

## Startup Order

Before making any claim or edit in this repo:

1. Read [CLAUDE.md](CLAUDE.md) — repo scope and the canonical-vs-speculative
   distinction.
2. Read [FLYWHEEL.md](FLYWHEEL.md) in full before summarizing or extending it.

## Verification

- This doc describes systems that live elsewhere (the DGX Spark machines /
  `ai-business` project). If asked whether something here is still true,
  verify against the live system rather than assuming the doc is current.
- Do not repeat the speculative valuation/acquisition claims referenced in
  CLAUDE.md as if they were verified facts. If asked about "the Disney
  discovery," describe it as an unverified agent narrative, not a business
  conclusion.

## Editing Rules

- Keep changes small and dated. Update "Last Updated" in `FLYWHEEL.md` when
  you change it.
- No secrets, tokens, or live IPs — placeholders only, even though this repo
  is private.
- This is a docs-only repo — don't add code, credentials, or copies of live
  system state here.
